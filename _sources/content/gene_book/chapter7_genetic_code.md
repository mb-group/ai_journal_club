# Chapter 7: Using the Genetic Code

## Mutations and Genetic Variants

### Protein Language Models for Mutations

#### MutaPLM: PLM with Mutation Explanation
- **Description**: Protein language model with mutation effect explanation
- **Input**: Protein sequences with mutations
- **Output**: Mutation effect predictions and explanations
- **Paper**: NIPS 2024
- **Application**: Understanding mutation impacts on protein function

### Genetic Variants and Expression

#### Genetic Variants Associated with Cell-type-specific Intra-individual Gene Expression
- **Description**: Analysis of genetic variants affecting gene expression
- **Input**: Genetic variants and expression data
- **Output**: Cell-type-specific expression associations
- **Paper**: Available
- **Application**: Personalized medicine, disease genetics

#### GPN-MSA: Multiple Sequence Alignment for Genetic Prediction
- **Description**: MSA-based genetic prediction network
- **Input**: Multiple sequence alignments
- **Output**: Genetic predictions
- **Paper**: Nature Biotechnology 2025
- **Application**: Evolutionary genetics, functional prediction

### SNP Foundation Models

#### SNP Foundation Model
- **Description**: Foundation model for single nucleotide polymorphisms
- **Input**: SNP data
- **Output**: SNP effect predictions
- **Paper**: Available
- **Application**: Genome-wide association studies, disease prediction

- **Description**: BERT model for codon usage analysis

- **Paper**: Available
- **Application**: Translation efficiency, codon optimization

#### CALM: Codon Adaptation and Language Model
- **Description**: Codon adaptation language model
