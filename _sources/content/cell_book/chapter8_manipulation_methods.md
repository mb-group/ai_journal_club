# Chapter 8: Manipulating Proteins, DNA, and RNA

## Designer Recombinases for DNA Editing

### RecGen: Designer Recombinase Generation
- **Description**: Conditional VAE for generating designer recombinases for DNA editing
- **Input**: Target DNA sequences and recombination requirements
- **Output**: Designer recombinase sequences
- **GitHub**: Available
- **Paper**: Nature Communication 2022
- **Core Technology**: Conditional Variational Autoencoder (VAE)
- **Training**:
  - Input: Recombinase sequences and target specificity data
  - Output: Trained generative model for recombinase design
- **Model Usage**: Generate novel recombinases for specific DNA editing tasks
- **Application**: Genome editing, synthetic biology, therapeutic applications
